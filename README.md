# Hyprland Macro Player

An application that executes keyboard macros in hyprland window manager. It can send keystrokes to background windows without switching focus.

Unlike existing Wayland macro recorders/players this one does not require any privileges or a background daemon.

This is a tauri app with a React frontend.
